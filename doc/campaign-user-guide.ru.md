# Taris Campaign Agent — Руководство пользователя

**Версия:** 1.0 · **Дата:** 2026-04-13  
**Аудитория:** Опытные пользователи — знакомые с Taris, N8N и Google Sheets  
**Тема:** Как запустить AI-рассылку по выбранной группе клиентов  

---

## 1. Что такое Campaign Agent?

**Campaign Agent** — это автоматизированный рабочий процесс в Taris, который позволяет отправить персонализированное письмо выбранной группе клиентов — прямо из чата Telegram. Вы описываете тему сообщения, а система:

1. Читает список клиентов из **Google Sheets**
2. Использует **AI (OpenAI GPT-4o-mini)** для выбора подходящих клиентов и составления персонализированного шаблона письма
3. Показывает **предварительный просмотр**, чтобы вы могли проверить и отредактировать текст перед отправкой
4. Отправляет письма через **Gmail** и записывает результаты обратно в Google Sheet

Для запуска рассылки не требуются технические навыки — только приложение Telegram и подготовленная Google Таблица.

### Компоненты системы

| Компонент | Что делает |
|---|---|
| **Taris (Telegram-бот)** | Ваш интерфейс управления — вы взаимодействуете с ним в Telegram для запуска и подтверждения рассылок |
| **N8N** | Система автоматизации, которая оркестрирует весь рабочий процесс (читает таблицу, вызывает AI, отправляет письма) |
| **Google Sheets** | Ваша база данных клиентов и журнал рассылок |
| **OpenAI GPT-4o-mini** | AI-модель, которая выбирает подходящих клиентов и генерирует персонализированный шаблон письма |
| **Gmail** | Сервис доставки электронной почты для отправки писем рассылки |

## 3. Предварительные требования

Перед запуском рассылки убедитесь, что:

| # | Требование | Кто настраивает |
|---|---|---|
| 1 | У вас есть доступ к **Telegram-боту Taris** и вы являетесь авторизованным пользователем | Администратор |
| 2 | **Google Таблица** со списком клиентов настроена (см. §4 ниже) | Вы |
| 3 | N8N настроен с вебхуком кампании и учётными данными Google/Gmail | Администратор |
| 4 | Переменные `N8N_CAMPAIGN_SELECT_WH`, `N8N_CAMPAIGN_SEND_WH` и `CAMPAIGN_SHEET_ID` заданы в `bot.env` | Администратор |

> **Для администраторов:** Детали настройки описаны в `doc/howto_admin.md`. Если какая-либо переменная не задана, бот покажет:  
> _"⚠️ Campaign agent не настроен. Задайте N8N\_CAMPAIGN\_SELECT\_WH и N8N\_CAMPAIGN\_SEND\_WH в bot.env."_

## 4. Подготовка Google Таблицы

Google Таблица является основным источником данных о клиентах. Вы управляете ею напрямую — Taris и N8N читают из неё автоматически.

### 4.1 Требуемая структура таблицы

Ваша Google Таблица должна содержать **три вкладки** (точные названия важны):

#### Вкладка 1 — `Клиенты`

Это ваша база данных клиентов. Каждая строка — один клиент.

| Столбец | Описание | Пример |
|---|---|---|
| `Имя` / `ФИО` | Имя клиента (имя и фамилия) | Анна Иванова |
| `Email` / `Почта` | Адрес электронной почты (обязателен для отправки) | anna@example.com |
| `Телефон` | Номер телефона (необязательно) | +49 171 1234567 |
| `Тип` | Категория клиента или статус VIP | VIP / Обычный |
| `Интересы` | Темы, которые интересуют клиента | здоровье, спорт |
| `Компания` | Название компании (необязательно) | Sintaris GmbH |
| `Доп. инфо` | Любые дополнительные данные для AI-отбора | Куплен курс по питанию |

> **Советы:**
> - Названия столбцов могут быть на русском или английском — N8N сопоставляет их автоматически
> - Следите за чистотой адресов электронной почты (без пробелов, без опечаток) — некорректные адреса приведут к тихой ошибке
> - Чем подробнее заполнены «Интересы» и «Доп. инфо», тем точнее AI подберёт клиентов для вашей темы

#### Вкладка 2 — `Шаблоны` — *необязательно*

Если вы хотите предоставить собственные базовые шаблоны для определённых типов мероприятий, добавьте их здесь. N8N будет использовать их как отправную точку перед AI-персонализацией. Оставьте вкладку пустой, если хотите, чтобы AI генерировал шаблон с нуля.

| Столбец | Описание |
|---|---|
| `Тема` | Тема шаблона (например, «Вебинар», «Акция») |
| `Текст` | Базовый текст письма |

#### Вкладка 3 — `Статус рассылок`

Эта вкладка **записывается N8N автоматически** после каждой отправки кампании. Вы можете читать её для отслеживания результатов.

| Столбец | Описание |
|---|---|
| `Дата` | Дата и время кампании |
| `Тема` | Тема кампании |
| `Отправлено` | Количество успешно отправленных писем |
| `Ошибка` | Адреса с ошибкой доставки (если есть) |

### 4.2 Настройка доступа к таблице

Необходимо, чтобы N8N имел доступ к чтению и записи вашей таблицы:

1. Откройте Google Таблицу → нажмите **Поделиться**
2. Добавьте служебный аккаунт Google из N8N (ваш администратор предоставит адрес)
3. Выберите уровень доступа **Редактор** и нажмите **Отправить**

## 5. Пошаговый запуск кампании

### Шаг 1 — Откройте меню агентов

В Telegram:
- Нажмите **Главное меню** → прокрутите вниз → нажмите **🤖 Агенты**
- В меню агентов выберите **📧 Кампания**

### Шаг 2 — Введите тему кампании

Бот запросит:  
_«Введите тему кампании (о чём письмо?):»_

Напишите тему в свободной форме. Примеры:

- `Вебинар по здоровому питанию 20 апреля`
- `Специальное предложение для VIP-клиентов на апрель`
- `Приглашение на онлайн-презентацию нашего нового курса`

### Шаг 3 — Введите фильтры (необязательно)

Бот запросит:  
_«Введите необязательные фильтры (или - для пропуска):»_

Фильтры позволяют дополнительно сузить список клиентов. Примеры:

- `тип: VIP, интересы: здоровье`
- `компания: Sintaris GmbH`
- `язык: немецкий`
- Отправьте **`-`** (дефис), чтобы пропустить и охватить всех клиентов, соответствующих теме

> **Примечание:** Фильтры передаются AI вместе с темой. Даже без фильтров AI включит только тех клиентов, чьи интересы совпадают с темой вашей кампании.

---

### Шаг 4 — Ожидание AI-отбора (~10–30 секунд)

Бот отобразит:  
_«⏳ Выбираю клиентов через N8N… (может занять до 30 сек)»_

В это время:
1. N8N читает вкладку `Клиенты` из вашей Google Таблицы
2. OpenAI GPT-4o-mini анализирует профиль каждого клиента в соответствии с вашей темой и фильтрами
3. AI генерирует персонализированный шаблон письма
4. N8N возвращает отобранных клиентов и шаблон в Taris

> **Если клиентов не найдено:** Бот сообщит _«Клиентов, соответствующих теме, не найдено.»_ — уточните тему/фильтры и попробуйте снова, или добавьте больше клиентов в таблицу.

---

### Шаг 5 — Просмотр предварительного превью

Бот отобразит карточку предпросмотра:

```
📋 Предварительный просмотр кампании

👥 Выбрано клиентов: 12
Анна Иванова, Пётр Мюллер, Мария Гарсия, … (+9)

📝 Шаблон письма:
Уважаемый(ая) {name},

Приглашаем вас на вебинар «AI-инструменты для
вашего бизнеса» 20 апреля в 18:00 по CET.
...
```

У вас есть три варианта действий:

| Кнопка | Действие |
|---|---|
| **✅ Отправить кампанию** | Подтвердить и отправить немедленно |
| **✏️ Редактировать шаблон** | Изменить текст письма перед отправкой |
| **❌ Отмена** | Отменить кампанию |

---

### Шаг 6 — Редактирование шаблона (необязательно)

Если вы нажмёте **✏️ Редактировать шаблон**, бот покажет текущий шаблон и предложит отправить новую версию.

В тексте шаблона можно использовать следующие **плейсхолдеры** — при отправке они автоматически заменяются данными каждого клиента:

| Плейсхолдер | Заменяется на |
|---|---|
| `{name}` | Имя клиента (из столбца `Имя`) |
| `{company}` | Название компании клиента |
| `{interests}` | Интересы клиента |

**Пример шаблона:**

```
Здравствуйте, {name}!

Мы рады пригласить вас на наш ближайший вебинар
по здоровью и благополучию. Учитывая ваш интерес
к {interests}, мы уверены, что вам будет полезно.

Дата: 20 апреля 2026 г. · 18:00 по CET
Регистрация: https://sintaris.net/webinar

С уважением,
Команда Sintaris
```

После отправки нового шаблона вы увидите обновлённый предпросмотр. Нажмите **✅ Отправить кампанию**, когда будете готовы.

---

### Шаг 7 — Подтверждение и отправка

Нажмите **✅ Отправить кампанию** для начала отправки.

Бот отобразит:  
_«📤 Отправляю кампанию… Пожалуйста, подождите.»_

N8N отправляет письма по одному через Gmail и записывает каждый результат во вкладку `Статус рассылок`.

---

### Шаг 8 — Получение результата

По завершении бот покажет:

```
✅ Кампания завершена!

📤 Отправлено: 12 писем

📊 Статус рассылки в Google Sheets → [ссылка]
```

Нажмите на ссылку, чтобы открыть вкладку `Статус рассылок` и увидеть, какие письма были доставлены, а какие (если таковые имеются) не удалось отправить.

## 6. Схема кампании — Краткая справка

```
Вы (Telegram)               Taris Бот              N8N               Google / AI / Gmail
──────────────              ─────────              ───               ───────────────────
📧 Открыть меню Агенты →   Показать запрос темы
Ввести тему         →       Сохранить тему
Ввести фильтры      →       Запустить N8N select  →  Читать Клиенты →  AI отбирает клиентов
                    ←       Показать предпросмотр ←  Вернуть клиентов + шаблон
Просмотр
  [Редактировать]   →       Запросить новый текст
  Отправить текст   →       Показать обновлённый превью
  [Отправить]       →       Запустить N8N send    →  Отправить Gmail →  Gmail доставляет
                                                  →  Записать Статус →  Таблица обновлена
                    ←       Показать результат    ←  Вернуть счётчик
Открыть ссылку      ←       Просмотреть таблицу
```

## 7. Устранение неполадок

| Проблема | Что делать |
|---|---|
| _«Campaign agent не настроен»_ | Обратитесь к администратору — необходимо задать URL вебхуков N8N в `bot.env` |
| _«Клиентов, соответствующих теме, не найдено»_ | Сделайте тему более общей, уберите фильтры или добавьте больше клиентов в таблицу |
| _«Учётные данные Google устарели»_ | Обратитесь к администратору — необходимо переподключить **Google Sheets Account 3** в N8N (N8N → Настройки → Учётные данные) |
| _«Ошибка AI-сервиса»_ | OpenAI API может быть временно недоступен — подождите минуту и попробуйте снова |
| _«Ошибка отправки письма»_ | Учётные данные Gmail могут потребовать обновления — обратитесь к администратору |
| Письма отправлены, но часть клиентов отсутствует | Проверьте столбец `Ошибка` во вкладке `Статус рассылок` — чаще всего причина в неверных адресах электронной почты |
| Кампания занимает более 2 минут | Таймаут N8N — администратор может увеличить `N8N_CAMPAIGN_TIMEOUT` в `bot.env` (по умолчанию: 90 секунд) |

## 8. Рекомендации

- **Поддерживайте список клиентов в актуальном состоянии.** Регулярно удаляйте из таблицы неверные или отписавшиеся адреса электронной почты.
- **Сначала протестируйте на небольшой группе.** Добавьте столбец `тип: тест` для тестовых клиентов и используйте `тип: тест` в качестве фильтра при первой кампании.
- **Используйте DEMO-режим для обучения.** Администратор может включить `CAMPAIGN_DEMO_MODE=true` в `bot.env`, чтобы использовать демонстрационных клиентов вместо реальной таблицы — удобно для обучения новых пользователей.
- **Проверяйте вкладку со статусом после каждой кампании.** Ошибки доставки могут указывать на устаревшие адреса или проблемы с учётными данными Google.
- **Не запускайте две кампании одновременно.** Состояние кампании хранится на уровне пользователя: если вы начнёте новую кампанию, пока предыдущая ещё выполняется, предыдущая будет отменена.

## 9. Часто задаваемые вопросы

**В: Можно ли добавить дополнительные столбцы в таблицу клиентов?**  
О: Да. N8N читает все столбцы и передаёт их AI. Дополнительные столбцы (например, «Дата последней покупки», «Язык») помогают AI делать более точный отбор.

**В: Клиенты получат персонализированные письма или один общий шаблон?**  
О: В шаблоне есть плейсхолдеры (`{name}`, `{company}`, `{interests}`), которые заменяются данными каждого клиента индивидуально перед отправкой.

**В: Можно ли запланировать кампанию на определённую дату и время?**  
О: Напрямую из Taris нет — кампания стартует сразу после вашего подтверждения. Для отложенной отправки обратитесь к администратору N8N с просьбой добавить узел задержки в рабочий процесс.

**В: Сколько клиентов можно охватить за одну кампанию?**  
О: В Taris жёсткого ограничения нет. Практические лимиты определяются дневной квотой отправки Gmail (обычно 500 писем/день для обычного Gmail, 2 000/день для Google Workspace).

**В: Где находится меню «Агенты»?**  
О: Главное меню → прокрутите вниз → **🤖 Агенты**. Если вы его не видите, возможно, у вас недостаточный уровень доступа — обратитесь к администратору.

**В: Можно ли отменить после нажатия «Отправить кампанию»?**  
О: Нет. После подтверждения отправки N8N начинает работу немедленно. Вы можете остановить запуск новых кампаний, но письма, уже находящиеся в очереди, будут доставлены.

## 10. Справка для администраторов — Переменные конфигурации

> *Этот раздел предназначен для администраторов, настраивающих систему, а не для рядовых пользователей.*

| Переменная | Описание | Пример |
|---|---|---|
| `N8N_CAMPAIGN_SELECT_WH` | URL вебхука N8N для отбора клиентов | `https://n8n.example.com/webhook/abc123` |
| `N8N_CAMPAIGN_SEND_WH` | URL вебхука N8N для отправки писем | `https://n8n.example.com/webhook/def456` |
| `CAMPAIGN_SHEET_ID` | ID Google Таблицы (из URL) | `1jQaJZA4cBS2sLtE42zpwDHMn6grvDBAqoK_8Sp6PmXA` |
| `N8N_CAMPAIGN_TIMEOUT` | Максимальное время ожидания ответа N8N (секунды) | `90` |
| `CAMPAIGN_DEMO_MODE` | Использовать демо-клиентов вместо реальной таблицы | `true` / `false` |
| `CAMPAIGN_FROM_EMAIL` | Адрес отправителя для всех кампаний | `info@sintaris.net` |

Все переменные задаются в `~/.taris/bot.env` на сервере. После любого изменения перезапустите Taris:  
```bash
systemctl --user restart taris-telegram
```
