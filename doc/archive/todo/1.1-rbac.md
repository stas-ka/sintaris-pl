## Role Model

| Role | System Chat | Admin Panel | Dev Menu | User features |
|---|---|---|---|---|
| **Admin** | ✅ Read + config ops only (allowlist) | ✅ Full | ❌ No | ✅ Full |
| **Developer** | ✅ All ops incl. restart/deploy (allowlist + confirm) | ✅ Full | ✅ Full | ✅ Full |
| **User** | ❌ No access | ❌ No | ❌ No | ✅ Full |
| **Guest** | ❌ No access | ❌ No | ❌ No | ⚠️ Pending approval |

## Command Allowlists (current)

**`ADMIN_ALLOWED_CMDS`** — read-only + config ops:
`cat`, `ls`, `head`, `tail`, `grep`, `systemctl status`, `journalctl`, `df`, `free`, `ps`,
`uname`, `ping`, `curl` (GET only), `python3 -c`, `echo`, `date`, `uptime`, `hostname`,
`lsblk`, `lsusb`, `dmesg`, `id`, `whoami`, `env`, `pwd`, `find`, `stat`

**`DEVELOPER_ALLOWED_CMDS`** — extends Admin:
`systemctl restart/stop/start`, `cp`, `mv`, `rm`, `python3`, `pip3`, `git`,
`pscp`, `plink`, `nano`, `vi`, `sed`, `awk`, `tee`, `chmod`, `chown`,
`mkdir`, `touch`, `wget`, `tar`, `unzip`
