## §8.5 NiceGUI Integration

**Goal:** Replace Jinja2 templates with NiceGUI for richer interactivity (sliders, live data binding, drag-and-drop).

**Current:** FastAPI + Jinja2 + HTMX + Alpine.js (~25 MB RAM overhead on Pi 3 B+)  
**NiceGUI estimate:** ~60 MB RAM overhead

**Checklist:**
- [ ] Evaluate NiceGUI RAM footprint on Pi 3 B+ via `psutil` benchmark
- [ ] Prototype single page (e.g. Voice Opts toggles) in NiceGUI
- [ ] If viable: migrate pages incrementally behind `USE_NICEGUI` feature flag
- [ ] If not viable: document decision here and close this item
