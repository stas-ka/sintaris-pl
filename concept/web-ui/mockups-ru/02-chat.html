Напиши Python-функцию для подсчёта частоты слов в тексте
08:15
🤖

Вот функция на Python для подсчёта частоты слов:

from collections import Counter

def word_frequency(text: str) -> dict[str, int]:
    """Подсчёт частоты слов в тексте."""
    words = text.lower().split()
    words = [w.strip(".,!?;:\"'()[]") for w in words]
    return dict(Counter(words))

# Пример использования
text = "кот сидел на мате кот сидел"
result = word_frequency(text)
print(result)
# {'кот': 2, 'сидел': 2, 'на': 1, 'мате': 1}

Функция делает следующее:

  1. Приводит текст к нижнему регистру
  2. Разбивает на слова и убирает пунктуацию
  3. Подсчитывает частоту через Counter

🎧 «Расскажи мне про историю Берлина»

Берлин — столица Германии с богатой историей, которая насчитывает более 800 лет.

Город был основан в XIII веке как торговый пост на реке Шпрее. К XVIII веку он стал столицей Пруссии, а затем — объединённой Германии в 1871 году.

В XX веке Берлин пережил две мировые войны, разделение на Восточный и Западный секторы, строительство и падение Берлинской стены (1961–1989), и наконец — объединение Германии в 1990 году.

Сегодня Берлин — один из крупнейших культурных и технологических центров Европы.
