# Concept — Future Ideas & Design Artifacts

This directory contains future concepts, design explorations, mockups, roadmaps, and archived prototypes that are **not part of the current implementation** but are relevant for the project's evolution.

## Contents

| Path | Description |
|---|---|
| `web-ui/` | Web interface concepts, mockups, and roadmap (FastAPI, NiceGUI, multimodal UX) |
| `telegram_menu_bot_original.py` | Original monolithic bot prototype (archived; current implementation is in `src/`) |

## Rule

- Files here are **reference only** — not deployed to any target host.
- Do not add currently-running source code or infrastructure files here.
- When a concept moves to implementation, the artefact should be **moved** from `concept/` to the appropriate `src/`, `doc/`, or `deploy/` directory.
